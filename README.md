# Blackboy Luxury Website v3

This version includes all 6 major upgrades requested for the website prototype.

## Included upgrades
1. Product detail pages
2. Cart and checkout flow
3. Local CMS/admin dashboard prototype
4. Working local contact form handling
5. Scroll reveal and page transition animations
6. SEO / Open Graph / social meta tags

## Important
This is still a static front-end prototype.
- Cart, orders, product edits, and enquiries are stored in browser localStorage
- Contact form is local demo only
- Checkout has no payment gateway
- Admin is local demo only
